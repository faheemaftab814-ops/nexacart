# NexaCart

A production-ready architecture foundation for a premium e-commerce platform built with Next.js 15, TypeScript, Tailwind CSS, shadcn-style UI primitives, Framer Motion, and Lucide React.

## Included foundation

- App Router structure
- Global layout and theme support
- Responsive header and footer
- Homepage placeholders and section layout
- Design tokens, utilities, constants, types, hooks, services, and config
- SEO-ready metadata, robots, and sitemap scaffolding

## Development

```bash
npm install
npm run dev
```
